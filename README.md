# CSV Data Analysis - Task 5

This project performs simple data analysis on a CSV file using Python and Pandas.

## Features
- Load CSV file
- View first rows of data
- Calculate total sales by product
- Calculate total sales by region
- Display bar chart using matplotlib

## How to Run

1. Install libraries
   pip install pandas matplotlib

2. Run the script
   python analysis.py

This task helped me understand basic data analysis using Pandas.
